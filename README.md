# Quiz Game 🎮
A command-line quiz game built with Python featuring a persistent leaderboard system and replayable gameplay loop.

## Features
- Multiple choice questions
- Randomised question order
- Score tracking
- High score saved to file
- Input validation
- Replay option

## 🛠️ Tech Stack
- Python
- JSON 
- File Handling

## How to Run
1. Clone the repo
2. Run:
   python main.py
